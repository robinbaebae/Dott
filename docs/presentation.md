## Slide 1: 표지

**Dott**
AI 브랜드 비서

## Slide 2: 문제 정의

### "이거 다 당신 얘기 아닌가요?"

| 😵‍💫 | 콘텐츠는 ChatGPT, 일정은 노션, 광고는 메타 대시보드, 트렌드는 구글링... |
|---|---|
| ⏰ | 매일 **5개 이상의 툴**을 왔다갔다 하면서 시간을 쏟고 있다 |
| 💸 | 마케터를 뽑고 싶지만 아직 **그럴 여유가 없다** |
| 📊 | 데이터는 쌓이는데 **분석할 시간이 없다** |

> 혼자서 마케팅의 모든 것을 감당하고 있다면, 당신만 그런 게 아닙니다.

## Slide 3: 솔루션

### Dott = AI 브랜드 비서

**마케팅에 필요한 모든 것을 macOS 앱 하나에.**

| 📢 마케팅 전문가 | 🎨 디자인 전문가 | 🔍 리서치 전문가 | 🛠️ 서비스 빌더 |
|:---:|:---:|:---:|:---:|
| 카피라이팅 | 배너 제작 | 트렌드 분석 | 기능 기획 |
| 캠페인 기획 | Figma 자동 Push | 경쟁사 모니터링 | 시스템 설계 |
| SNS 전략 | 크리에이티브 디렉션 | SEO 키워드 | 기술 전략 |
| 이메일 마케팅 | 비주얼 브랜딩 | 데이터 기반 제안 | 버그 분석 |

## Slide 4: AI 활용 핵심 — 에이전트 파이프라인

### 3단계로 끝나는 워크플로우

```
┌─────────────┐     ┌──────────────────┐     ┌─────────────────┐
│  01. 말하세요  │ ──▶ │ 02. Dott이 판단합니다 │ ──▶ │ 03. 결과를 받으세요 │
│             │     │                  │     │                 │
│  자연어로     │     │  AI 오케스트레이터가  │     │  브랜드 맥락을     │
│  원하는 걸    │     │  최적의 전문가를     │     │  이해한 결과물이    │
│  말하면      │     │  자동 배정          │     │  나옵니다         │
└─────────────┘     └──────────────────┘     └─────────────────┘
```

**핵심: 사용자는 "누구에게 시킬지" 고민할 필요 없음**
→ AI 오케스트레이터가 메시지를 분석하여 최적 에이전트 자동 배정

## Slide 5: AI 활용 핵심 — 아키텍처 상세

### AI 에이전트 파이프라인

```
사용자 입력: "인스타 성과 분석해줘"
         │
         ▼
┌─── classifyTask() ───┐
│  자연어 → 에이전트 분류  │   API키 있으면: Claude AI 분류
│  + 필요 도구 판단       │   API키 없으면: 로컬 키워드 매칭
└──────────┬───────────┘
           ▼
┌─── 컨텍스트 수집 (병렬) ──┐
│  • 브랜드 가이드          │
│  • Instagram 데이터       │
│  • Meta 광고 데이터       │
│  • 메모 & 인사이트        │
│  • 웹 검색 (필요시)       │
└──────────┬───────────┘
           ▼
┌─── executeAgentTask() ──┐
│  선택된 전문 에이전트 실행   │   Claude API / CLI
│  + 수집된 컨텍스트 주입    │
└──────────┬───────────┘
           ▼
┌─── 후처리 ──────────────┐
│  배너 생성 / 블로그 초안   │
│  캘린더 등록 / 이메일 작성  │
│  Figma 자동 Push         │
└─────────────────────────┘
```

## Slide 6: 주요 기능 (1/2)

### 14개 기능이 하나의 앱에

| 기능 | 설명 |
|------|------|
| **AI 대화** | 물어보기만 하면 Dott이 알아서 판단. 인스타 성과 분석, 광고 카피 작성 등 |
| **콘텐츠 워크스페이스** | 기획부터 발행까지 — 블로그, SEO 브리프, 이메일 시퀀스, 캘린더 |
| **광고 분석** | Meta 광고 데이터 자동 수집 + AI가 ROAS, 지출, 노출 인사이트 제공 |
| **트렌드 & 리서치** | RSS 피드, 경쟁사 모니터링, 키워드 트렌드 자동 수집 |
| **인사이트 & 메모** | 링크/아티클/영상 스크랩, AI 대화에서 자동 참조 |
| **프로모션 빌더** | 할인/쿠폰/번들 프로모션 설계 + AI 카피 생성 |
| **Figma 자동 Push** | AI가 생성한 배너를 Figma 캔버스에 자동 배치 |

## Slide 11: 데모 모드 안내

### API 키 없이도 모든 기능을 체험할 수 있습니다

Dott은 **완전한 데모 모드**를 내장하고 있어, 별도 설정 없이 바로 시연 가능합니다.

**데모에 포함된 콘텐츠:**
- 8개 샘플 태스크 (인스타 릴스, 광고 카피, SEO 키워드 등)
- 일일 AI 활동 보고서
- 14개 알림 (배너 생성, 태스크 완료, 트렌드 갱신 등)
- 6개 콘텐츠 캘린더 항목
- 5개 키워드 트렌드 (AI마케팅 +145%, 숏폼전략 +89%)
- 10개 최신 트렌드 아티클
- Instagram 9포스트 + Threads 9포스트
- 9가지 AI 대화 시나리오 (배너, 카피, 트렌드, 블로그, 이메일 등)

**진입 방법:** 랜딩페이지 → "데모 체험하기" 버튼

> 랜딩페이지: https://dott-blush.vercel.app
